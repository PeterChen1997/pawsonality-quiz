# 爪格实验室：宠物人格测试数据说明

> 本文档说明本项目如何基于 15 维向量进行匹配，以及本次品牌化重构后的内容结构。

## 项目定位

本项目基于 `pingfanfan/SBTI` 二次开发，保留了：
- 纯前端静态部署方式
- 15 维度 + 曼哈顿距离匹配逻辑
- 特殊人格彩蛋机制

重构后重点改成：
- 面向中文互联网传播的 **宠物 MBTI / 狗格测试 / 猫格测试**
- 更适合女性主理人做营销验证的轻内容语气
- 可配置的主理人 / 品牌露出位
- 页面内明确标注源码来源

## 评分模型

### 1. 维度结构

共 15 个维度，分为 5 个模型：

1. 本体气场
2. 情感亲密
3. 生活态度
4. 行动节奏
5. 社交边界

每个维度 2 道题，题目选项分值为 1 / 2 / 3。

### 2. 分数计算

- 每个维度两题求和，得到范围 `2 ~ 6`
- 分段映射为：
  - `2~3 => L`
  - `4 => M`
  - `5~6 => H`

### 3. 人格匹配

把 15 个维度转成一个 `L/M/H` 向量，并与人格库中的 pattern 做曼哈顿距离比较：

- 距离越低越相似
- 若距离相同，则优先 exact 命中更多者
- 再按 similarity 排序

### 4. 特殊人格

- `SPIN`：微醺贴贴怪
  - 当饮酒彩蛋问题触发时覆盖普通结果
- `MIXI`：混种魅力体
  - 当标准人格相似度低于阈值时作为兜底

## 品牌化内容结构

### 首页
- 品牌名
- 宠物人格测试副标题
- 主理人信息
- 源码来源说明

### 结果页
- 主人格
- 15 维雷达图
- TOP 5 匹配列表
- 主理人露出位
- 源码来源说明
- 品牌化免责声明

### 分享图
- 结果 code + 中文名
- 匹配度
- 15 维雷达 / 条形图
- 主理人露出
- 品牌 footer

## 可继续定制的最关键文件

- `data/config.json`：品牌名、主理人、部署命令、免责声明
- `data/questions.json`：题目语气和营销导向
- `data/types.json`：结果人格命名与描述
- `src/style.css`：视觉风格
